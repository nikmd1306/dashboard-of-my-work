# Work Dashboard

Персональный дашборд для людей, которые совмещают несколько видов рабочей деятельности — фриланс-проекты, основную работу, свой бизнес и всё, что между ними.

Вместо того чтобы переключаться между таймтрекером, финансовым приложением и таск-менеджером, Work Dashboard даёт одно место, где видна полная картина: куда уходит время, сколько приносит каждое направление и что требует внимания прямо сейчас.
